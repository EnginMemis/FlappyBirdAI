# FlappyBirdAI

This project has been created to learn deep reinforcement learning. </br>

## Train your own model

```bash
python agent.py
```

## If you have a checkpoint model

```bash
python agent.py --checkpoint "model/model.pt"
```

## Test the model

```bash
python test.py --model "model/model.pt"
```
